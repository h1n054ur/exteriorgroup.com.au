# Project Overview - Exterior Group

## Executive Summary
Exterior Group is a service-oriented website (Commercial & Residential cleaning/maintenance) currently running on a legacy static/CMS-exported stack. The project is undergoing a modernization effort to move to a high-performance stack using **Bun**, **Hono**, and **HTMX**.

## Tech Stack Summary

| Category | Technology | Version | Note |
| --- | --- | --- | --- |
| Type | Legacy Static | - | CMS-exported |
| Frontend | HTML5 / JS | - | Static files |
| Styling | Bootstrap | 5.3.2 | CDN & Local |
| Icons | Font Awesome | - | - |
| Analytics | GTM | - | Google Tag Manager |

- **Architecture**: Static Feature-based folders
